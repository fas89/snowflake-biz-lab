# Code Of Conduct

This project is intended to be a practical, welcoming place to learn and improve
the Snowflake Telco Lab.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still naming technical risks clearly.
- Keep discussion focused on the work, the users, and the safety of the release.
- Be careful with credentials, customer-like examples, and security reports.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Publishing private information or credentials.
- Dismissing security concerns without investigation.
- Sustained disruption of issues, pull requests, or community spaces.

## Enforcement

Project maintainers may remove comments, close threads, or restrict
participation when behavior makes the project unsafe or unproductive.

For security issues, follow `SECURITY.md` rather than opening a public issue.
